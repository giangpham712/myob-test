Assumptions:
- Employee name can contain any characters
- Employee name that contains more than one word should be enclosed in double quotes

How to run:
- Run the program using Visual Studio (Input needs to be set in Application arguments)
- Run using command line
  + Build the solution (Run "dotnet build" in solution root directory)
  + Change directory to MYOB.CodingTest
  + Run "dotnet run"

How to run tests
- Run the tests in MYOB.CodingTest.Tests project using Visual Studio
- Run using command line
  + Run "dotnet test" in solution root directory or MYOB.CodingTest.Tests directory
